## Getting Started

First, clone the repository and move to the file

```sh
  git clone https://github.com/pablogiaccon/tv-shows/

  cd tv-shows
```

### Installation

1. Install NPM packages
   ```sh
   npm install
   # or
   yarn
   ```
2. Execute the project
   ```sh
   yarn start
   ```
3. Using the mobile Expo Client scan the QRCode
